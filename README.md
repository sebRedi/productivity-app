# Productivity App 🚀

## Project Title

Productivity Management System

## Getting Started

This project is a full-stack productivity application that allows users
to manage tasks, projects, and reminders.

## Prerequisites

-   Java 17+
-   Maven
-   Node.js
-   npm

## Installing

### Backend

``` bash
cd productivity-app
mvn clean install
mvn spring-boot:run
```

### Frontend

``` bash
cd productivity-frontend
npm install
npm run dev
```

## Running tests

``` bash
mvn test
```

## Deployment

Backend runs on: http://localhost:8080

Frontend runs on: http://localhost:5173
